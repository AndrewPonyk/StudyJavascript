Notes:
1) <select> is easy var single=$("#single").val(); - getting selected item
2) <select multiple="multiple"> selector : var multiple=$("#multiple").val(); alert(multiple.join(",")); - getting selected items
3) be carefull with checkboxes : var checkbox=$("#c:checked").val(); checkbox=checkbox|| "off"; - without this if checkbox is not selected the value will be undefined
4) with radio is easy var radio=$("input:radio[name=r]:checked").val(); - get value of selected radio
